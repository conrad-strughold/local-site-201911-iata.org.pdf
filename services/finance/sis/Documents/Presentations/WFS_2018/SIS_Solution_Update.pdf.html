Access IS-WEB via recent web browsers: Why? 



Only recent web browser versions are getting patched with latest 

security fixes 



Don’t take a chance to be exposed 



SIS is leveraging on recent browsers capabilities 



e.g. in SIS Rel. 1.13, one change (CMP#743 - Upload Multiple Input Files from 

Screen Upload File), so if you don’t have a recent web browser version

…  this feature will simply not work !
