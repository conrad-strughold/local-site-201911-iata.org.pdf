Downtime for SIS Major Release 1.14 Deployment 

Deployment of SIS Major Release 1.14 on the SIS Production environment will begin on Wednesday May 8th, 2019 

at 00:00 Eastern Standard Time or 04:00 UTC and will last for approximately 4 hours.  

 

IS-WEB will be unavailable during this time and all files submitted via iiNET will be processed after the downtime has 

been completed. 



 

SIS Browsers -  Important Security Update 

A  recent  audit  performed  under  IATA  security  control  revealed  that  several  SIS  users  are  still  accessing  SIS  with 

outdated browsers versions or operating systems which are not supported any more hence creating possible security 

risks. 

In order to keep the platform safe and not expose SIS and its integrated ICH solution, it is mandatory for all users 

concerned to proceed to a browser upgrade to the most recent version by contacting their IT department. Similar to 

the exercise conducted in August 2018, SIS access will be restricted with a reinforced browser control and only the 

below browsers will be accepted from the date of implementation: 



 

Chrome v60 and above 



 

Internet Explorer v11 (i.e. blocking all previous IE versions like IE9 / IE10 )  



 

Edge v13 and above 



 

Firefox v60 and above  



 

Safari v12 and above 

./services/finance/sis/Documents/Bulletins/SIS-Communication-2019-May-html.html

A list of users using older versions was identified and those users will be advised individually via a direct email to take 

immediate  action.  More  generally,  all  SIS  users  are  requested  once  more  to  validate  if  they  are  using  a  recent 

browsers version as to keep security as its best.  

 

We strongly encourage you to take the necessary steps to check with your relevant IT department to validate that 

your SIS access is made using a recent browser version either from Chrome, Internet Explorer, Firefox or Safari. 

 

The new restriction for SIS IS-WEB access will be implemented on 24-Jun-2019 with SIS Release 1.14.2.0 and if 

you  have not proceeded to the recommended upgrade at that time, your SIS access will be restricted (you will not 

be able to login).
