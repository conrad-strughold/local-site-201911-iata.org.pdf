SIS Major Release 1.15 

– Changes Deployment in SIS Sandbox 

Changes for SIS Major Release 1.15 will be deployed on the SIS Sandbox on Tuesday October 8

th

, 2019.  There are no 

new validations introduced with this major release nevertheless, we encourage all Members to submit test files to 

the SIS Sandbox environment to ensure compliance, prior to the deployment of the changes on SIS Production. You 

may consult SIS Release Notes for Major Release 1.15
