SIS Major Release 1.14 

– Update to CMP 755 “Addition of Passenger Rejection Reason Codes” 

This  CMP  will  introduce  6  new  reason  codes  to  the  existing  list  of  Passenger  Rejection  Reasons  to  allow  future 

rejections to be more specific and give additional information to the airline receiving the rejection.  

 

The proposed reason codes for Involuntary Reroutes “3F”, “3G” and “3H” already exist for use in Passenger Sampling 

UAF rejections. Therefore, new reason codes have been assigned as follows. 

 

Reason Code 1Z: Involuntary Rerouting Fare Difference 

Reason Code 1T: Involuntary Rerouting Tax Difference 

Reason Code 1R: Involuntary Rerouting Fare and Any other Difference 

 

We request  members to  be advised of this update. All other newly introduced reason  codes of  this CMP remain 

unchanged.
