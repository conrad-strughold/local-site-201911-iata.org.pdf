CMP 669 - CSV listings for MISC Invoices/Credit Notes 

This document aims to provide guidance and principles for members to benefit from 

the  new  SIS  feature  implemented  in  Release  1.11:    “

CSV  listings  for  MISC 

Invoices/Credit Notes

”.  

The new feature provides a new Listing in CSV format in addition to the existing PDF 

listing for Miscellaneous Invoices/Credit Notes. 

The purpose of this new feature is to allow any billing parties to receive a file from SIS 

in addition to the traditional Invoice and PDF listing as to facilitate some automation 

for  invoices  reconciliation.  The  new  CSV  listing  has  a  standardized  format  and 

structure for all charge categories or type of billings allowing Members to import such 

information  locally  (e.g.  into  a  spreadsheet)  and  perform  semi-automated 

reconciliation, sorting of fields per flight or per service as required, for instance.

How to enable CSV Listings option 

 
Members can enable this option through following steps: 

Step 1) Navigate to Member Profile >> ‘Miscellaneous’ tab >> ‘CSV Listings’ section  

 

Step 2) Click on checkbox ‘CSV listing required as a Billing Entity’ and/or 'CSV listing 

required as a Billed Entity’  

The new CSV listing will be included in 

Payables OAR

 only when 'CSV listing required 

as a Billed Entity’ is checked in member profile. 

The new CSV listing will be included in 

Receivables OAR

 only when 'CSV listing required 

as a Billing Entity’ is checked in member profile.

 

Step  5)  Click  on  ‘

Save  Miscellaneous  Details

’  in  ‘Miscellaneous’  tab  to  save  the 

changes.

How to download the CSV Listing 

The new CSV Listing file is located under the listing folder 

 

The  CSV  Listing  file  will  contain  all  fields  for  all  Charge  Categories  and  its  format  / 

structure for all the Charge Categories is standardized and identical
