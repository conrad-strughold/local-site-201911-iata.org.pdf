SIS Quick Tips - IS-WEB Navigation  

 

Problem:  
 
When a user navigates through a large list of items in a search results grid (or within an invoice) the list 
is split on multiple pages. If the user selects to view an item on a specific page and then clicks on the 
Back button (eg. Back To Billing History) from within the item, the system automatically take the user 
back to page 1 of the item list instead of the page they were previously on. 
 
Solution: 
 
To remain on the specific page the user can view the item in a new browser tab by following these 
steps.  
 
1.  From the Search Results grid 

 

 

2.  Right Click on the view button 

 and 

select “Open link in new tab” 

 

 

 
(Using keyboard shortcut Ctrl+Left Click also works)  

 
 
3.  This opens a new browser tab with the item the user wanted to view and also keeps the original tab 

open with the specific page from the item list the user was viewing.
