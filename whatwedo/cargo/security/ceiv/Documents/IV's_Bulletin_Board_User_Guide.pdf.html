1  Introduction to the Bulletin Board 

1.1  Purpose 

The  independent  validator’s  bulletin  board  lists  the  Regulated  Agents  (RA3)  and  Known 
Consignors (KC3) which have been assessed as compliant with the regulation EU 185 2010 and 
its related amendments by Independent Validators. The information in this board originates from 
Independent Validators and has last been updated on the date shown above. 

 

The 

independent validator’s bulletin board is available on line

. 

This service is free of charge. 

 

Important disclaimer

: Please note that IATA does not warrant that the information contained in 

the bulletin board is fully up-to-date. The board cannot be considered an official nor exhaustive 
list of the entities designated by Independent Validators. Please refer to the 

terms for using this 

board

. 

 

1.2  Definitions 

ACC3:  Air  Cargo  or  Mail  Carrier  operating  into  the  Union  from  a  Third  Country  Airport,  as 
defined in EU regulation 185/ 2010. Since 1st July 2014, a carrier must undergo an independent 
validation in order to keep its ACC3 designation. 

 

CEIV - Center of Excellence for Independent Validators. The CEIV was created by IATA to train, 
advise and support industry stakeholders complying with EU regulation 185/ 2010. 

 

EU: European Union. 

 

EU regulation 185/ 2010: This EU regulation requires the "air cargo or mail carrier operating into 
the Union from a third country airport" to gain an ACC3 designation. As of 1st July 2014, an air 
carrier's  designation  only  continues  to  be  valid  upon  successful  completion  of  an  EU  Aviation 
Security Validation performed by an Independent Validator accredited by an EU member state. 
Only validated carriers are authorized to fly cargo or mail into Europe. This regulation foresees 
as well RA3 and KC3 status. 

 

IV - Independent  Validator: In the context of ACC3 Regulation:  Individual accredited by an EU 
member states to perform ACC3/ RA3/ KC3 independent validations. 

 

KC3:  A  known  consignor  is  a  consignor  who  originates  cargo  or  mail  for  its  own  account  and 
whose procedures meet common security rules and standards sufficient to allow the carriage of 
cargo  or  mail  on  any  aircraft.  A  KC3  designates  a  known  consignor  that  complies  with  the

RA3: A regulated agent is an agent, freight forwarder or any other entity who conducts business

2.2  Searching in the bulletin board 

A number of search criteria are available in the bulletin board. 

 

You can use any combination of these to search for organizations in the bulletin board. 

Most of the search criteria have mechanisms to ease entry by the user. Typically drop down lists 
are proposed. Note that these drop down lists only include values for which there is at least one 
entry  in  the  board.  For  instance  the  drop  down  list  on  the  search  criteria 

Country

  only  shows 

countries for which there is at least one entry in the database. In case a country is not proposed 
in the list, this means that there is no entry for this country in the board yet. 

The  search  field 

Company

  does  not  use  a  drop  down  list.  Instead,  the  system  will  return  any 

record  for  which  the  company  name  contains  the  string  that  has  been  entered  in  this  search 
field.  As  an  example,  searching  for 

abc

  returns  all  companies  whose  name  include  the  string 

abc

. 

The  search  result  list  shows  the  entities  fulfilling  your  search  criteria.  The  left  columns  identify 
the activity of the company, its name and the location(s) where it is active:



  It  is  possible  for  the  same  company  to  have  several  activities  (for  instance  acting  as  a 

GHA and as a Screening Agent). 



  It is possible for the same company to be active at several locations (for instance at the 

airports LHR and LGW, 2 airports in London area) 



  In case an organization has several activities and / or several locations, it might happen, 

depending on the search criteria that have been provided, that several lines of the result 
list relate to the same company (each line of the result list relates to one company, one 
activity of this company, one location of this company) 

 

When performing a search, only the first 30 entries of the board fulfilling your search criteria are 
displayed  in  the  search  result  list.  This  is  to  ensure  a  quick  response  time.  It  is  at  any  time 
possible to include additional search criteria to refine your search.

3  Help with the Bulletin Board 

If  you  have  issue  using  the 

independent  validator’s  bulletin  board,  please  contact  the  IATA 

CEIV: 

ceiv@iata.org

 

If you are an independent validator and would like to report RA3 and KC3 into the bulletin board, 
please contact the IATA CEIV: 

ceiv@iata.org

 

If you have questions regarding the data documented in the bulletin board for your organization, 
please contact the IV who supplied this data into the bulletin board.
