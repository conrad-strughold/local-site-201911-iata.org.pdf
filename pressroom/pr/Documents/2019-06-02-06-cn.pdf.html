纸质旅行证件的情况下，在机场各个环节快速办理手续。

出示纸质护照或登机牌，每位旅客都会喜欢这样的便捷性。

国际航协的

 One ID

决议要求利益相关方（包括航空公司、机场和政府机构）通力合作，利用生物识别技术

来促进和实施无纸化旅客流程。

 

航空公司呼吁：

 

•

 

国际民航组织（

ICAO

）成员国支持数字旅行证书规范，该规范可提供安全有效的护照替代方案。

 

•

 

航空业与各国政府密切合作，制定全球标准，旨在根据各种数据隐私要求和法规，在需要知情

/

允

许知情的基础上，安全可靠地访问、传递和使用旅客身份信息。

 

One  ID

计划的愿景是无纸化旅行体验，旅客只需提供个人生物识别数据即可安全环游世界。这一美好愿景

可通过可靠的数字身份、生物识别技术和各种授权利益相关者访问的协作身份管理平台来实现。

 

 

目前，世界各地的机场正针对国内机场流程，测试

One ID

理念的各个要素，其中包括：
