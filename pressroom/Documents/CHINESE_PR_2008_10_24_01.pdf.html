术语说明：

 

o 

RPK

：收入客公里 (

Revenue Passenger Kilometres

) 计量实际的旅客运输量

 

o 

ASK

：可用座位公里 (

Available Seat Kilometres

) 计量可用的旅客运输能力

 

o 

PLF

：客座率 (

Passenger Load Factor

) 是可用座位公里的使用比率。将 2007 年与

2006 年相比时，

PLF

代表两个相比较时期之间的点差异

 

o 

FTK

：货物吨公里 (

Freight Tonne Kilometres

) 计量实际的货物运输量

 

o 

ATK

：可用吨公里 (

Available Tonne Kilometres

) 计量可用的总运输能力(客运和货运

合计)
