DISH revision 22.0 in order that ‘C’ becomes a valid value for the ‘Data Input 

Status Indicator

’ DISI data element, as it currently is published in the latest DISH 

Revision 23. 
 
As a result, the please note the new DISH DISI entry below that is to be reflected within 
DISH Revision 22 and used in conjunction with Update Bulletins 2016/1 and 2018/1.

Description

 

 

An indication of the mode of data capture. 

 

Use

 

 

This element is only applicable for refund transactions. 

 

Values

 

 

A 

Fully automated data and manually captured 
cancellation penalty/refund charge and waiver code of 
cancellation penalty 

C 

Fully automated data and amounts calculated using 
refund rules filed by the carrier 

F 

Fully automated (no agent entered data is included in 
the Refund transaction) 

L 

Mixed automated data and data manual captured 

M 

Manual data capture 

– no automated data 

 

RET Validation 

 

If TRNC = ‘RFND’, then DISI must contain “A”, “C”, “F”, “L” or “M”.
